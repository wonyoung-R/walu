# 🕹️ 찌릿 (chrit)

> 동료랑 아이스크림 내기 하자 — **24가지 한판 미니게임**. 웹 · 모바일 어디서나, 설치 없이 바로 플레이.

레트로 CRT 감성의 미니게임 모음 페이지입니다. 한 판이 짧고, 기록이 남고, 친구한테 결과를 복사해서 도발할 수 있어요. 서버도 로그인도 없는 **정적 HTML 파일 하나**라 GitHub Pages로 무료 배포됩니다.

🔗 **라이브 데모:** `https://<your-id>.github.io/arcade/` _(배포 후 이 줄을 본인 주소로 바꿔주세요)_

---

## ✨ 특징

- **비용 0원** — 백엔드·DB·빌드 과정 없음. 정적 파일 하나로 끝.
- **웹 + 모바일** — 포인터/터치 모두 지원, 반응형 레이아웃.
- **승부욕 자극** — 게임마다 기기 내 최고기록 저장, 신기록 연출, 결과 복사해서 도발하기.
- **중독성** — 한 판이 5~30초, "다시하기" 한 번이면 즉시 재도전.
- **칩튠 사운드** — Web Audio로 생성하는 8비트 효과음 (음소거 토글 있음).
- **오프라인 친화** — 외부 의존성은 구글 폰트뿐. 기록은 브라우저 `localStorage`에 저장.

## 🕹️ 조작

- 데스크톱: 마우스 클릭 / 키보드 `Esc`로 메뉴 복귀
- 모바일: 탭 · 드래그 · 스와이프
- 상단 우측 `♪` 버튼으로 사운드 켜고/끄기

## 🚀 GitHub Pages 배포

1. GitHub에서 새 저장소 생성 (예: `arcade`, Public).
2. `index.html` (과 `README.md`)를 업로드하고 커밋.
3. **Settings → Pages → Build and deployment**에서 Source를 `Deploy from a branch`, Branch를 `main / (root)`으로 지정 후 Save.
4. 1~2분 뒤 `https://<your-id>.github.io/arcade/` 에서 접속.

> 파일이 하나뿐이라 폴더 구조가 필요 없고, 수정 후 다시 올리거나 `push`하면 1~2분 내 자동 반영됩니다.
> 개인 도메인은 **Settings → Pages → Custom domain**에서 연결할 수 있어요.

## 📣 후원 / 링크

메뉴 그리드 중간중간에 게임 카드와 같은 크기의 후원 카드가 배치되어 있습니다.

- 📸 인스타그램: [@mavs.kr](https://www.instagram.com/mavs.kr)
- 🏀 홈페이지: [mavs.kr](https://mavs.kr)

> 링크/문구/노출은 `index.html`의 `PROMOS` 배열로 조절합니다. 배열의 각 항목이 그리드에 **딱 1장씩**, 게임 사이에 고르게 배치됩니다(현재 인스타·홈 2장).
> 항목을 추가하면 그만큼 후원 카드가 늘고, 빼면 줄어듭니다. (예: 항목 3개 → 후원 카드 3장이 균등 분배)

## 🛠️ 기술

- 순수 **HTML / CSS / JavaScript** (프레임워크 없음)
- **Canvas 2D** — 자유투 · 양궁 · 리프팅 · 줄넘기 · 과일 슬라이스
- **Web Audio API** — 칩튠 효과음
- **localStorage** — 게임별 최고기록 저장 (실패 시 메모리 폴백)

## 📝 메모

- 기록은 **접속한 기기·브라우저에만** 저장됩니다. 서버 동기화·계정 기능은 없습니다.
- 시크릿 모드나 일부 환경에서 `localStorage`가 막히면 그 세션 동안만 기록이 유지됩니다.
- 새 게임은 `GAMES` 배열에 `mount(root, api)` 객체를 추가하고 `INFO`에 설명을 넣으면 메뉴에 자동 등록됩니다.
